MyCineLog - Film Takip Sistemi
Bu uygulama, Java Swing kullanılarak geliştirilmiş bir film yönetim sistemidir. Kullanıcıların izledikleri veya izlemek istedikleri filmleri dijital bir liste halinde tutmalarını sağlar.

--Özellikler
Film Kayıt Sistemi: Yeni filmleri sisteme ekleme ve mevcut listeyi görüntüleme.

Veri Yönetimi: Veriler, kalıcı depolama için filmler.txt dosyası üzerinde saklanır (Java File I/O).

Görsel Arayüz: Kullanıcı dostu ve modern bir Java Swing tasarımı.

🛠 Kullanılan Teknolojiler
Dil: Java

Veri Saklama: Metin Dosyası (txt)

Geliştirme: Cursor AI / VS Code
